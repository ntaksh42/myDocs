# .NET 10 概要

## リリース情報

.NET 10は**2025年11月11日**に正式リリースされました。本バージョンは**Long-Term Support (LTS)** リリースとして提供され、**2028年11月14日まで3年間**のパッチとアップデートのサポートが保証されています。

## 主な特徴と改善点

### パフォーマンスの向上

.NET 10では、パフォーマンスに関する大幅な改善が実現されています：

- **.NET 8と比較して平均レスポンスタイムが49%高速化**
- **JSONシリアライゼーションが20-40%高速化**
- JITインライニングの改善
- メソッドの非仮想化の強化
- スタック割り当ての最適化
- AVX10.2サポート
- NativeAOTの機能強化
- 構造体引数のコード生成改善
- ループ反転の最適化強化

### AI統合

.NET 10では、AI機能の統合が大きな焦点となっています：

- **Microsoft Agent Framework**の導入
- Azure SQLおよびSQL Serverでの**ベクトル検索サポート**
- **Retrieval-Augmented Generation (RAG)** などの高度なAIワークロードの有効化
- AI関連の拡張機能とツールの強化

### C# 14の新機能

C# 14では、よりクリーンで保守しやすいコードを実現する新機能が導入されました：

- **フィールドバックプロパティ (Field-backed properties)**
- **新しいSpan変換**
- **Null条件付き代入演算子**
- **コレクション式の拡張**
- **部分コンストラクタ (Partial constructors)**
- **ref struct インターフェースサポート**

### 暗号化とセキュリティ

`System.Security.Cryptography`ライブラリに、3つの新しい非対称ポスト量子アルゴリズムのサポートが追加されました：

- **ML-KEM (FIPS 203)**
- **ML-DSA (FIPS 204)**
- **SLH-DSA (FIPS 205)**

これらは量子コンピュータ時代に備えたセキュリティの強化を提供します。

### ASP.NET Core 10の更新

ASP.NET Core 10.0では、以下の新機能と改善が導入されました：

#### Blazorの改善
- Blazor WebAssemblyのプリロード機能
- 自動メモリプール削除
- フォームバリデーションの強化
- 診断機能の改善

#### その他の機能
- **OpenAPI機能の強化**
- **Minimal APIの更新**
- **Passkey（パスキー）サポート**によるIdentity認証の強化

### 開発者エクスペリエンス

開発者の生産性を向上させる新機能：

- **単一ファイルベースのアプリケーション作成**が可能に
- プロジェクトファイルなしで`dotnet run`コマンドを使用して単一の`*.cs`ファイルを実行可能
- プログラムの作成と実行が簡素化

### Entity Framework Core 10

EF Core 10では、以下の機能強化が行われました：

- **LINQの機能強化**
- **パフォーマンス最適化**
- **Azure Cosmos DBのサポート改善**
- **名前付きクエリフィルタ**：エンティティタイプごとに複数のフィルタを設定し、選択的に無効化が可能

### .NET MAUI の更新

.NET Multi-platform App UI (MAUI) の主な改善点：

- **MediaPickerの機能強化**：複数ファイルの選択と画像圧縮
- **WebViewリクエストインターセプト**
- **Android API レベル35および36のサポート**

## SDK とツール

.NET 10 SDKには、開発ワークフローを改善する多数の機能強化が含まれています：

- MSBuildの改善
- ビルドパフォーマンスの向上
- 診断ツールの強化

## まとめ

.NET 10は、LTSリリースとして、そして.NETエコシステムへの大規模なパフォーマンスおよび機能アップデートとして、重要なマイルストーンを表しています。AI統合、パフォーマンスの大幅な向上、C# 14の新機能、ポスト量子暗号のサポートなど、現代のアプリケーション開発のニーズに応える包括的なアップデートとなっています。
